# Real-Time Chat Platform

A production-grade, real-time chat system built using WebSockets, message queues,
caching, and cloud-native DevOps practices.

## Key Features
- Real-time messaging with WebSockets
- Fault-tolerant message delivery
- Horizontal scalability
- Cache-optimized performance
- 12-Factor compliant design
- CI/CD-ready architecture

## Tech Stack
- Node.js
- WebSockets
- Redis
- Message Queue (Kafka/RabbitMQ)
- Docker & Kubernetes
